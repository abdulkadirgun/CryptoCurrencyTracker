## Tech Stack
- Hilt
- Coroutines & Flow
- Jetpack Navigation
- Retrofit & Gson
- Firebase Authentication
  - register
  - login
  - logout
  - check logged in
- Firebase Cloud Store
  - create
  - read
  - update
  - delete
- Room Database
- Glide
- WorkManager
- In-app Notifications
